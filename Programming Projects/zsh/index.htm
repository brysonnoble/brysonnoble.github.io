Oh My Zsh Smirnoff Theme


I created this theme as a quick and fun side project while working on the final project for one of my classes. The theme was inspired by and named after the Smirnoff drink, and actually ended up looking quite nice. The demo above is just a quick web-based simulation of what it looks like to run in terminal, but the code for the theme is below.


FLAG_COLOR_1="white"
FLAG_COLOR_2="blue"
FLAG_COLOR_3="red"
PATH_COLOR="yellow"
FULL_PATH_COLOR="red"
BRACKET_COLOR="white"

PROMPT='%{$fg[$FLAG_COLOR_1]%}■%{$fg[$FLAG_COLOR_2]%}■%{$fg[$FLAG_COLOR_3]%}■%{$reset_color%} %{$fg[$PATH_COLOR]%}%1~ %{$fg[$FLAG_COLOR_1]%}➧ '
RPROMPT='%{$fg[$BRACKET_COLOR]%}[ %{$fg[$FULL_PATH_COLOR]%}%~%{$fg[$BRACKET_COLOR]%} ]%{$reset_color%}'

You can use it yourself by visiting my fork of Oh My Zsh, an open-source framework for managing your zsh configuration, created and maintained by Robby Russell, Marc Cornellà, Larson Carter, and Carlo Sala.
